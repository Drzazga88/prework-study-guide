# Prework Study Guide

## Description

This guide contains basic information about HTML, CSS, JS and GitHub. It was built as a first project and it was not only a flavour to coding but actual working guide which can be used to store tips and tricks.

## Usage

Each section contains notes about a certain topic, like HTML, CSS, JavaScript, Git. And if we open the DevTools console, we can see the JavaScript running. It will list the topics we learned from prework and choose one topic for us to study first.
